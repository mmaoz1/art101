Lab 4 - Pseudocoding and Problem-Solving

Challenge

The challenge of this lab is to practice psuedo code and problem solving. We are tasked with breaking down complicated tasks into small, incremental steps. Both for a real life task and a simple computer game.

Problems

The challenge for me in this lab was to break down 'rock, paper sciscors' as a comptuer game and make it into small steps. I'm not too familiar with this type of coding so I took a shot in the dark on what would actually need to be done in order to write this code.

Reflection

This assignment was helpful for learning how to create an outline for coding and problem solving. I was much more familiar with my first set of Instructions on how to record music. Thus my list was much more extensive. On the other hand, my instructions for creating an online rock, paper, scissors game were shorter, since I really don't even know where to start.
